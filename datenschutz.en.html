General

The operator itself does not process the values entered into the calculator; they remain exclusively in the user's browser. No registration or sign-in is required. When the website is loaded, however, technically necessary connection data is processed by the hosting provider (Vercel Inc.) — see the "Hosting" section below for details. The operator does not perform any personal evaluation of these connection data.

Local Data Storage

Values entered by the user — such as consumption, prices, and mileage — are stored exclusively in the browser's local storage. This data does not leave the device and is not transmitted to or processed by the operator at any time.

Service Worker / Offline Cache

This website uses a Service Worker — a standardised browser feature that caches HTML pages and static resources (CSS, JavaScript, fonts, images) in the user's browser. The sole purpose is faster revisits and offline availability of the app.

The Service-Worker cache holds only static assets (HTML, CSS, JavaScript, fonts, images) and does not store calculator inputs. Beyond the connection data already processed by the hosting provider when those assets are originally requested, the cache itself does not transmit anything; it lives in the user's browser and can be cleared at any time via the browser's site settings ("Clear browsing data" or similar) — the only effect is that the next visit no longer benefits from cached assets.

Hosting

This website is hosted by Vercel Inc. When the page is loaded, technically necessary connection data — in particular IP address, user agent, timestamp, requested resource (request URL) and referrer — are processed by the hosting provider to deliver the website, defend against attacks and ensure service stability. The legal basis is Art. 6(1)(f) GDPR (legitimate interest in the secure operation of the website). Vercel acts as a processor within the meaning of Art. 28 GDPR; the Data Processing Agreement is part of Vercel's Terms.

Vercel retains these server logs for a maximum of 30 days in accordance with standard practice and deletes them automatically afterwards.

The operator does not directly link this connection data to identifiable individuals.

Technical and Organizational Measures

To protect data processed via this website, the following technical and organizational measures pursuant to Art. 32 GDPR are implemented:

  • Encrypted transmission of all data between browser and server via HTTPS (TLS 1.3)
  • Strict Content Security Policy (CSP) to prevent cross-site scripting and unauthorized third-party scripts
  • No server-side processing of calculator inputs by the operator: all calculations occur exclusively in the user's browser
  • No tracking, analytics, or advertising technologies
  • No cookies (features such as market or history storage operate exclusively via the browser's LocalStorage without data transmission)
  • Regular updates of libraries and frameworks in use
  • Hosting with an ISO 27001 and SOC 2 Type II certified provider (Vercel Inc.)

Beyond the infrastructure security mentioned above, the operator's safeguards consist in avoiding any data collection that goes beyond what is technically necessary.

No Cookies

EVSpend does not set any cookies. Functional state — your selected market, theme, saved calculations and similar preferences — is held purely in your browser's localStorage. No identifier is transmitted to our servers and no value crosses an HTTP request.

Your initial market is determined once based on the IP-geolocation provided by our hosting provider (Vercel) for the page request. If that result does not match your preference, you can switch markets via the in-app market selector at any time; the selection is then remembered locally.

You can clear localStorage at any time via your browser's site settings; doing so resets the app to its initial geo-derived state on the next visit.
