Accessibility

Accessibility statement under German BFSG / WCAG 2.1 AA

Scope

This statement applies to the web application EVSpend, available at www.evspend.com, including all subpages and the corresponding mobile presentation (Progressive Web App).

Conformance Level (Self-Assessment)

Based on internal review, this web application is broadly aligned with WCAG 2.1 Level AA (corresponding in substance to BITV 2.0 / EN 301 549). No external conformance audit or certification has been performed; individual deviations are possible and will be corrected once identified.

Semantic HTML, ARIA labels for interactive elements, and full keyboard operability are implemented. The additional measures for screen readers and contrast requirements are documented below.

Implemented Measures

According to our self-assessment, the known core functions are broadly oriented around the WCAG 2.1 Level AA criteria. This is an internal review; no external audit or certification has taken place. Implemented measures include:

• Slider input fields with dynamically updated aria-valuetext information for screen readers
• Screen reader announcements via aria-live region when a slider is released
• Color contrast per WCAG 1.4.3 Contrast Minimum (normal text ≥ 4.5:1) — including the hero result values (EV / combustion) in light mode, which since the update of 28 April 2026 use dedicated text tokens (--ev-text #15803d, --orange-text #b45309) instead of the brand-accent shades
• UI elements per WCAG 1.4.11 Non-Text Contrast (≥ 3:1)
• Historical chart with textual alternative: summary text as figcaption plus complete data table for screen reader table navigation
• Market-aware and language-aware text alternatives for all three supported markets (DE/US/TR)
• Units and currency names spelled out in screen reader text (to avoid inconsistent symbol pronunciation)

The classification is based on internal review and has not been certified by an external accessibility auditor.

How This Statement Was Prepared

The information is based on a self-assessment by the operator against WCAG 2.1 Level AA criteria, not on external certification. This statement was prepared on April 24, 2026 and last updated on May 1, 2026 (conformance wording softened to a self-assessment).

Voluntary Disclosure

This statement is provided voluntarily. As a private individual, the operator is not legally required under current rules to publish an accessibility statement under the German Accessibility Act (BFSG). The information is provided for transparency and to facilitate communication.

European Accessibility Context

The European Accessibility Act (EAA, Directive (EU) 2019/882) entered into force on June 28, 2025 and references requirements for digital services to be accessible to persons with disabilities. EVSpend's accessibility work is oriented around WCAG 2.1 Level AA and requirements commonly referenced by the EAA. This statement does not assert formal compliance with or certification under the EAA.

Practices applied:

  • WCAG 2.1 Level AA criteria used as orientation
  • Keyboard navigation throughout the calculator
  • Screen-reader-friendly slider labels and announcements
  • Color contrast reviewed against WCAG AA contrast thresholds
  • Reduced-motion preference respected

If you encounter accessibility barriers, please contact us at g.uer@aol.com with details. We aim to respond within four weeks.
